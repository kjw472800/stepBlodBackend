## Development
```bash
1. npm install
# config environment variables 
2. set your.env
DB_HOST=mongodb+srv://sudo:<password>@cluster0.la1t2.mongodb.net/<dbname>?retryWrites=true&w=majority \
SALT_ROUND=YOUR_SALT_ROUND 
jwtKey=YOUR_JWTKey 
GOOGLE_API_KEY=YOUR_GOOGLE_API 
# run dev server 
3. npm start
```
